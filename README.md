# james-portfolio

Personal portfolio website for James Prabu.

## Project

This repository contains a single static HTML portfolio page:

- `james_prabu_portfolio_final.html`

## Run locally

Open the HTML file directly in your browser:

1. Navigate to this repository folder.
2. Double-click `james_prabu_portfolio_final.html`.

Or use VS Code Live Server if you prefer a local server.

## Deployment

This project is suitable for GitHub Pages, Netlify, or any static hosting provider.
